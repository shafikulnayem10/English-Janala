# 📘 English-Janala

**English-Janala** is a modern and beginner-friendly English learning web platform.  
It helps users learn vocabulary easily through lessons, search functionality, and interactive UI.

---

## 🚀 Live Site

🔗 **Live Link:**  
https://shafikulnayem10.github.io/English-Janala/

---

## 📌 Project Features

- 🔐 Simple Login Authentication (Email & Password Based)
- 📚 Lesson-based Vocabulary System
- 🔎 Search Functionality for Words
- 🧠 Word Details Modal View
- ⏳ Loading Spinner for Better UX
- 📱 Fully Responsive Design
- 🎨 Clean UI using Tailwind CSS & DaisyUI

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Tailwind CSS  
- DaisyUI  
- JavaScript (DOM Manipulation)
